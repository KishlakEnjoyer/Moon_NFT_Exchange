# Moon_NFT_Exchange
Moon_NFT_Exchange — дипломный проект: децентрализованная платформа для обмена Telegram-подарков на Ethereum/Polygon. Реализованы смарт-контракты (Solidity), интерфейс (Vue), интеграция с MetaMask, аукционы и хранение метаданных через Supabase. Полнофункциональная тестовая версия для защиты.
